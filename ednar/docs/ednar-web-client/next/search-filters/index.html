Search Filters

After the search criteria button has been pressed the Filter Configuration panel will appear which will allow Basic or Advanced searching of created NARs.

Basic​

1. The State of the NAR can be searched by selecting one of the options in the list that appears when the State field is selected.

2. The Access Type

3. The Switching Type of the NARs can be searched. When the Switching Type search field is selected a list of options will appear.

4. The Filter configuration allows for the Start Date Time to be searched. When the Start Date Time field is selected a calender will appear and the date can be chosen.

5. The Filter configuration allows for the End Date Time to be searched. When the End Date Time field is selected a calender will appear and the date can be chosen.

6. The Participants of the NARs can be searched. The Participants field will show options for participants once letters have been entered.

7. The Planner assigned to the NAR can be searched using this field. The Planner field will options for planners once a letter value has been entered.

8. The Planning Area assigned to the NAR can be searched by clicking on the Planning Area field and selecting the area from the list that appears.

8. The Control Desk assigned to the NAR can be searched by clicking on the Control Desk field and selecting the area from the list that appears.

10. The Add Questions button opens a panel that allows search filtering by the questions answered in the NARs.

Advanced​

1. The Select field drop down menu allows the user to select search criteria. Once one is selected, the user is prompted to Select Comparator and enter more information as required.

2. The + Rule button allows the user to add a new rule.

3. The + Group button allows the user to add a new group.
