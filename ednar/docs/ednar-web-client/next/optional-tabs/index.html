Optional Tabs

note

The exact wording of the triggering questions will differ between organisations

LV Switching​

To add this tab add the question "Is LV switching required?" to the Application Tab's questions section. Then select yes and LV Switching will become available.

Once the LV Switching tab has been selected the Suggested LV Switching section will have an Add Step button that must be selected to an LV Switching Steps.

When the steps have been added Actions and Apparatus can be selected in a series of steps. These steps can be rearranged or deleted as required. At the bottom of the tab is a section for attacking an LV Map/Diagram. This can be attached by selecting the attachment type from the drop down list then selecting a file and uploading it to the NAR.

HV Generators​

To add this tab add the question "One or more HV generators?" to the Application Tab's questions section. Then select yes and the HV Generators tab will become available.

After selecting the HV Generators tab a Generator will need to be added. This can be done by clicking the blue Add Generator button.

The HV Generator section can be filled out by first selecting a generator option from the drop down list that will appear when the Generator Option field. The Estimated generator load can be added by entering a value into the relevant field. Under the Rating field a generator rating can be selected from a drop down list. The Location of generator connection can be entered into that field. The next section allows for the diagrams of the generator supplied area to be attached by selecting an attachment type from the drop down list then selecting and uploading a file. The final field in the HV Generator section allows for Additional information to be added.
