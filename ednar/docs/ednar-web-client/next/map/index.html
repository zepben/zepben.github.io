Map

Clicking the Map feature in the creation of a new planned request will open a map. It will be possible to annotate and take screenshots of this map.

1. The Map view can be used to search for assets on the map. To do this, enter the asset's name into the search box.

2. This section allows for Layers to be selected and deselected. The tick boxes can enable and disable features overlayed on the map.

3. This sections allows for different modes to be chosen. allowing for asset selection, snapshots, and annotations.

The snapshot mode has a camera tool that takes a screenshot of the map which is then attached to the NAR under the name entered when the snapshot is taken.

Annotate mode is the other default mode that can be chosen.

If the Annotate mode is chosen the annotation tools will become available.

The polygon tool allows a shape to be constructed by using the tool to select the corners of the shape.

The circle tool allows a circle to be constructed by selecting a centre point for the circle and enlarging the circle by moving the mouse. The circle is set by clicking the mouse.

The line tool operates similarly to the polygon tool. Points can be created by clicking on the map and a line will connect the points. An exit button will replace the colour picker while using line mode. This button must be selected to end the line that is created.

The callout tool creates as two point line similar to the line tool. However, the second point on the callout tool will have a box that will accept text. Once typed the tick at the bottom of the text box must be clicked and the callout will then appear on the map.

The colour picker allows the colour of the lines, circles and polygons to be changed.

When a change is made to the map the discard change button will appear. If this is clicked the change last made will be removed.

After a change is made to the map, that change can be clicked on. A delete icon will then appear allowing for any annotations to be removed.

If the authority type is chosen the Asset Selection mode will be available. Asset Selection allows for the Isolation Points and Apparatus To Be Covered to be chosen directly from the map.
