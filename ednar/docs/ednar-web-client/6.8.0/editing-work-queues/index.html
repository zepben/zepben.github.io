Editing Work Queues

After adding the desired search criteria to the search work queue. Though the process outlined under Search Filters. The work queues can be edited and saved for use later if desired.

The plus button allows for the creation of the work queue this will open a name panel that allows for the created work queue to be named.

This button allows search criteria in the work queue to be changed using the search filter configuration panel.

The search button refreshes the search.

The close button closes the selected work queue and closes the search.

This button saves the current search criteria to the work queue.

This button deletes the work queue so that it cannot be chosen from the work queue list.

Copying the current work queue for use in any other work queue can be done by clicking the copy button.
