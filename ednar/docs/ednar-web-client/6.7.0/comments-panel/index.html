Comments Panel

The Comments Panel allows for comments on the NAR to be written and read. All comments made include time stamps and the name of the commenter.

To mention another user in a comment, type in the @ key and start typing the user's name or username. A list of users will display in a dropdown - select one of these to include them in the comment, and email notifications can be configured to send to these mentioned users.
