Basic Searches

The basic search allows a user to specify filters on many of the fields in the NAR. The filters on all fields must match for a NAR to be returned in the search.

The screen shot below shows the Basic filter with some filters applied, and some explanatory text around how the filters work.

ednar_user_manual_basic_search

These filters are turned into a Boolean logic expression, and this can be viewed by clicking on the “More” button just under the logic expression viewer, as illustrated in the screenshot below.

Clicking More will bring up the following view, which is showing the Boolean expression that is used for the Filter.

More information on how the logic of the filter works can be found in the following section.
