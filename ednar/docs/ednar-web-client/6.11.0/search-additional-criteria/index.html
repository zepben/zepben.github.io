Additional Search Criteria

At the bottom of the NAR search filter configuration panel there are three collapsible Additional Criteria panels. The first is Access Authority. The second Job Title, and the third is Customer Notification.

1. The Isolation Points of the NARs can be searched by entering a value into the Asset Name field as with creating a new planned request options will appear under the Asset Name field to be choosen from.

2. The Apparatus To Be Covered can be searched by entering a value into the Asset Name field which will create a list of options that can be chosen from.

3. A Primary Feeder can be search for by entering its name into this search field. The tick box at the side can be selected and any NARs that do not have entered primary feeders can be searched.

4. The Location of the NAR can be searched by entering it into the Location field. The tick box at the side can be selected and any NARs that do not have a set location can be searched.

5. The Work Title can be searched by clicking on the search field and selected the Work Title from the list that appears. The tick box at the side can be selected and any NARs that do not have a selected Work Title can be searched.

6. PON State

7. PON Start Status

8. Has Customer Notification provides three options to search the NARs, Yes, No or Unanswered can be selected and the search will be filtered by whether the customer has been notified.

9. Yes, No or Unanswered can be selected for Checked Outage Viewer to search by the selected answer, or lack of answer.

10. Clicking on the Customer Interruption Off Time field will open a calendar. From this calendar a date can be choosen and NARs with this off Time will be search. The search can also be refined by time by clicking the select time option.

11. Clicking on the Customer Interruption On Time field will open a calendar. From this calendar a date can be choosen and NARs with this off Time will be search. The search can also be refined by time by clicking the select time option.
