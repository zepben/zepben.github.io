Access Types

Configurable access types for the NAR.

Table Name: ACCESS_TYPES

ColumnData TypeDescription
IDNUMBERUnique ID
NAMEVARCHAR2(128)Unique access type name
LIFE_CYCLEVARCHAR2(128)Use one of either NAR_DMS_SCHEDULE, NAR or NAR_SIMPLE. More information on each can be found in NAR Lifecycles.
CUSTOMER_NOTIFICATION_OPTIONVARCHAR(100)Determines what answers are available to the "Customer Notification Required?" NAR question. Available options are NONE to disable the question, NO to lock the answer as No, YES to lock the answer as Yes and SELECT to allow the applicant to choose. Default is SELECT.

NameDescriptionDefault Value

NOTIFICATION_SMS_DOMAINSMS domain to be used for sending SMS notifications.
INVALIDATED_SNAPSHOT_CHECKBoolean value that determines whether the client should check for invalidated snapshots.true
EXTERNAL_MAP_ASSET_VIEW_URLUrl template for viewing assets on an external map application.

The following supported template tags will be substituted with appropriate values on the client when viewing an asset on an external map application: <X>, <Y>, <Z> : Longitude, Latitude and Elevation.

For eg: For an arbitrary asset, the following template url https://address/?x=<X>&y=<Y>&z=<Z> will be formatted as http://address/?x=10&y=20&z=2
EXTERNAL_MAP_OUTAGE_VIEW_URLUrl template for viewing outage on an external application.

The following supported template tag will be substituted with an appropriate value on the client when viewing an outage on an external application: <PON_ID>: PoN Project ID.

For eg: For an arbitrary interruption, the following template url https://address?outage=<PON_ID> will be formatted as http://address?outage=10
PON_PRINT_DATE_OFFSETNumber of business days that must be used as an offset (# days before the outage date) for the default print date of a PoN Project8
LONG_OUTAGE_THRESHOLD_HOURSNumber of hours for a customer notification task duration after which the user is prompted for confirmation8
HANDCARD_THRESHOLD_DAYSNumber of business days before a customer notification task outage after which the delivery method must be Handcard.8
EIC_THRESHOLD_DAYSNumber of business days before a customer notification task outage after which the delivery method must be EIC4
NAR_REPORT_PERMITS_IS_AVAILABLEBoolean value that determines whether the Permits option is available for NAR Reports in the UIfalse
NETWORK_MAPS_DEFAULT_LATDouble value that specifies the default network map location latitude in the UI0.0
NETWORK_MAPS_DEFAULT_LNGDouble value that specifies the default network map location longitude in the UI0.0
NETWORK_MAPS_DEFAULT_ZOOMInteger value that specifies the default network map zoom level in the UI0
NETWORK_MAP_SOURCESComma-separated array of string values that specifies the available network maps in EDNAR. Available options are EWB for the EDNAR native map and PEEK for the Peek DMS Diagram viewer. The first item in the array will be the default map that opens when selecting assets from the NAR.None
ASSET_STRING_FORMATString value that specifies the format of assets representation when typing or select the assets in the UI e.g want to change the order of representation or change the separator or add some extra string({{#equipmentContainers}}{{name}}{{#seperator}}, {{/seperator}}{{/equipmentContainers}}) {{#voltage}} - {{.}} {{/voltage}} {{#assetName}} - {{.}} {{/assetName}} {{#poiText}}{{name}}{{#seperator}}, {{/seperator}}{{/poiText}} {{#description}} - {{.}} {{/description}}
POLE_NAME_STRING_FORMATString value that specifies the format of pole assets in the UI{{description}} [{{name}}]
TOTAL_NMI_COUNTInteger value of the total customers on the network. If this app option is set to anything other than 0, a SAIDI Minutes estimate will be displayed for customer notification tasks0
MAPBOX_API_KEYString value of the Mapbox API token that EDNAR will use for its geo maps. To generate a free token, you can visit https://account.mapbox.com/access-tokens/, create an account and assign all the public scopes to the token. The token string should start with "pk".
SWITCHING_SCHEDULE_QUICK_VIEW_INCLUDE_PERMITSBoolean value that determines if the switching schedule "quickview" will include permitsfalse
NAR_CALENDAR_PILL_FORMATDefines the information to display in the NAR pills on the calendar. Available tags are:
- {{id}} - NAR ID
- {{jobTitle}} - Work title from NAR title
- {{jobType}} - Job Type/Switching Type for the NAR
- {{equipmentContainer}} - Primary equipment container for the NAR
- {{location}} - Location of the NAR from the NAR title		{{id}}
ENABLE_SYNTHETIC_ASSETBoolean value that enables the client to add synthetic assets to NARs via freetext entryfalse
ASSET_SEARCH_AVAILABLEBoolean value for enabling/disabling asset searchtrue
