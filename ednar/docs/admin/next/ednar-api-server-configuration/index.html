EDNAR API Server Configuration

This page describes how to configure and run the EDNAR WEB API Server.

Prerequisites​

  • Java 11 or 17. Java 21 has not been tested for support with EDNAR.
  • If using Oracle, a user already created with permissions to create schema objects and run various DDL statements.

Configuration​

Configuration for the EDNAR server is specified in a JSON file.

Command Line Arguments​
FlagValid ValuesRequiredDefault
-confPath to the configuration fileYes

To run the server, execute the following command at a terminal:

java -jar <path to ednar jar> -c <path to config file>
