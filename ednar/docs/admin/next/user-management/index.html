User Management

Privileges​

Privileges can be viewed using the admin app by navigating to the User Management page from the top menu.

Roles​

Roles can be viewed using the admin app by navigating to the User Management page from the top menu.

To create a role use the Create Role button next to the search bar on the right. Role can be edited by using the View action against a role.

A role needs to be assigned a name and an optional description.

The transfer component can be used to assign privileges by transferring available privileges on the left to the right column using the arrow.

The UI will indicate if there are any unsaved changes. Once the changes have been made, it can be saved using the Save button on the top right.

To delete a role use the Delete action against a role item and select Ok on the confirm dialog.

Users​

User info can be viewed using the admin app by navigating to the User Management page from the top menu.

To select a user search using the user's name and select the relevant option from the list.

The transfer component can be used to assign roles to a user by transferring available roles on the left to the right column using the arrow. User can also be marked as a planner and an operator.

The UI will indicate if there are any unsaved changes. Once the changes have been made, it can be saved using the Save button on the top right.
