EDNAR OMS Service

Introduction

info

The implementation of the EDNAR OMS Service is bespoke, as its operation requires integration with other corporate systems.

EDNAR OMS Service runs as a windows service to synchronize customer notifications between EDNAR and OMS database at a specified time interval.

Customer notifications will be saved in OMS database by executing a stored procedure named save_nar_customer_notifications, passing in sql type nars_typ. See SQL Type section for the type structure.

The software runs as a windows service. It is built using .NET and can run only on windows.

Required Software

  • Atleast Windows 7 operating system with SP1 or above is required
  • .NET Framework 4.6.2

Installing EDNAR-OMS Service

note

You need administrator privileges to install the service.

  • Login to the windows machine in which EDNAR-OMS Service needs to be installed.
  • Create a folder EdnarOmsService.
  • Copy all the EDNAR-OMS Service binaries from the .zip file of the release to this folder.
  • Change the config file to make sure it is correct in the running environment.
  • Run Powershell as an administrator and use it to navigate to the EdnarOmsService directory.
  • Run .\ednar-oms-service.exe install and wait for a message to say that it was successfully installed.
  • You should get the installed successfully message.
  • The EDNAR-OMS Service windows service is installed at this point.
  • Go to the Services console in windows.
  • Make sure the EDNAR-OMS Service service is running. If not, start it.
  • Check the properties of the EDNAR-OMS Service service to ensure that it is configured to start automatically.

Uninstalling EDNAR-OMS Service

note

You need administrator privileges to uninstall the service.

  • Login to the windows machine in which EDNAR-OMS Service is installed.
  • Run Powershell as an administrator.
  • Navigate to C:\EdnarOmsService.
  • Run the command .\ednar-oms-service.exe uninstall
  • You should get an uninstall successful message.

Configuring Ednar-OMS Service

  • Navigate to C:\EdnarOmsService.
  • Open the file ednar-oms-service.exe.config in the text editor of choice.
  • Update the configuration items as specified below. Save and close the file. Restart EDNAR-OMS Service for the new configuration to take effect.
<add key="DBConnectionRetries" value="2" />
<add key="RunInterval" value="60" />
<connectionStrings>
    <add name="EdnarOracleDB" connectionString="Data Source=(DESCRIPTION=(ADDRESS=(PROTOCOL=TCP)(HOST=192.168.30.129)(PORT=1521))(CONNECT_DATA=(FAILOVER_MODE=(TYPE=select)(METHOD=basic))(SERVER=dedicated)(SERVICE_NAME=nms)));User Id=username;Password=********;" />
    <add name="DMSOracleDB" connectionString="Data Source=(DESCRIPTION=(ADDRESS=(PROTOCOL=TCP)(HOST=192.168.30.129)(PORT=1521))(CONNECT_DATA=(FAILOVER_MODE=(TYPE=select)(METHOD=basic))(SERVER=dedicated)(SERVICE_NAME=nms)));User Id=username;Password=********;" />
    <add name="PonOracleDB" connectionString="Data Source=(DESCRIPTION=(ADDRESS_LIST=(ADDRESS=(PROTOCOL=TCP)(HOST=192.168.30.129)(PORT=1521)))(CONNECT_DATA=(SERVICE_NAME=nms)));User Id=username;Password=********;" />
</connectionStrings>
Config ItemDescription
DBConnectionRetriesNumber of retries the service will use to try to connect to the database before giving up.
RunIntervalThe amount of time in seconds that the service will sleep before running a new task.
Connection String - EdnarDBSpecify the connection string for the EDNAR database in oracle.
Connection String - OmsDBSpecify the connection string for the OMS database in oracle.

SQL Type

-- Customer Notification Template
CREATE OR REPLACE TYPE template_typ AS OBJECT (
   id NUMBER(38,0),
   name VARCHAR2(256)
);

-- Customer Notification Task NMI
CREATE OR REPLACE TYPE nmi_typ AS OBJECT (
   nmi VARCHAR2(256)
);
CREATE OR REPLACE TYPE nmis_typ AS TABLE OF nmi_typ;


-- Customer Notification Event Reason Type
CREATE OR REPLACE TYPE customer_notification_event_reason_typ AS OBJECT (
   id NUMBER(38,0),
   name VARCHAR2(256),
   type VARCHAR2(128)
);

-- Customer Notification Event
CREATE OR REPLACE TYPE customer_notification_event_typ AS OBJECT (
   id NUMBER(38,0),
   template template_typ,
   reason customer_notification_event_reason_typ,
   created_on TIMESTAMP(6),
   nmis nmis_typ
);
CREATE OR REPLACE TYPE customer_notification_events_typ AS TABLE OF customer_notification_event_typ;

-- Customer Notification
-- Customer Notification Task
CREATE OR REPLACE TYPE customer_notification_task_typ AS OBJECT(
   company VARCHAR2(128),
   description VARCHAR2(256),
   outage_start TIMESTAMP(6),
   outage_end TIMESTAMP(6),
   template template_typ,
   delivery_method VARCHAR2(256),
   nmis nmis_typ,
   events customer_notification_events_typ
);
CREATE OR REPLACE TYPE customer_notification_tasks_typ AS TABLE OF customer_notification_task_typ;

-- NAR Access Type
CREATE OR REPLACE TYPE access_type_typ AS OBJECT(
   id NUMBER(38,0),
   name VARCHAR2(128)
);

-- NAR Job Type
CREATE OR REPLACE TYPE job_type_typ AS OBJECT(
   id NUMBER(38,0),
   name VARCHAR2(128)
);

-- User
CREATE OR REPLACE TYPE user_typ AS OBJECT (
   username VARCHAR2(256),
   first_name VARCHAR2(128),
   last_name VARCHAR2(128),
   email VARCHAR2(256),
   fax VARCHAR2(32),
   depot VARCHAR2(64),
   mobile VARCHAR2(32),
   phone VARCHAR2(32),
   position VARCHAR2(128)
);

-- NAR Comment
CREATE OR REPLACE TYPE nar_comment_typ AS OBJECT (
    id NUMBER(38, 0),
    comment_timestamp TIMESTAMP(6),
    comment_user user_typ,
    type VARCHAR2(100),
    text VARCHAR2(4000)
);
CREATE OR REPLACE TYPE nar_comments_typ AS TABLE OF nar_comment_typ;

-- NAR Participant
-- Participant Type
CREATE OR REPLACE TYPE participant_typ AS OBJECT (
   participant_type VARCHAR2(128),
   participant_user user_typ
);
CREATE OR REPLACE TYPE participants_typ AS TABLE OF participant_typ;

-- NAR
CREATE OR REPLACE TYPE nar_typ AS OBJECT(
   id NUMBER(38,0),
   state VARCHAR2(128),
   access_type access_type_typ,
   job_type job_type_typ,
   start_date_time TIMESTAMP(6),
   end_date_time TIMESTAMP(6),
   primary_equipment_container VARCHAR2(128),
   work_title VARCHAR2(128),
   location VARCHAR2(128),
   participants participants_typ,
   customer_notification_tasks customer_notification_tasks_typ,
   comments nar_comments_typ
);
CREATE OR REPLACE TYPE nars_typ AS TABLE OF nar_typ
