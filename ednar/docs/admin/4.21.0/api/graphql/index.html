GraphQL API

EDNAR has a GraphQL accessible at POST /api/v1/graphql. Queries can be made to this API using GraphQL query syntax.

EDNAR's GraphQL API is largely self-documenting through the GraphQL introspection system. However, below is a list of available queries and mutations, their arguments, available fields, and an example query for each. This should provide a good starting point to using the GraphQL API.
