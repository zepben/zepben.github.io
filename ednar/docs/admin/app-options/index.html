App Options

This page provides further details on configuring the options available in the APP_OPTIONS database table.

Setting and Adjusting App Options

As the App Options are located in a database table, editing them is more complicated than other admin configuration. If you have a database GUI tool, you may be able to edit the table directly, otherwise you will need to use SQL statements.

To set the initial value for an app option, you will need to insert both the option and the value into the table. The following SQL can be used:

Oracle:
INSERT INTO app_options (name, value)
SELECT '<option name>', '<value>' FROM dual;

MS SQL Server:
INSERT INTO app_options (name, value)
SELECT '<option name>', '<value>';

To update an existing option value, you can use this SQL:

Oracle and MS SQL Server:
UPDATE app_options SET value = '<value>' WHERE name = '<option name>';

For eg: For an arbitrary asset, the following template url https://address/?x=<X>&y=<Y>&z=<Z> will be formatted as http://address/?x=10&y=20&z=2
