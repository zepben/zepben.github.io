EDNAR Web Client

The EDNAR Web Client is the single-page application (SPA) that provides the user interface that users interact with in their web browsers. The Web Client consists of static assets, meaning these can be served by the EDNAR App Server itself, or by something such as IIS.

This page contains information on the configuration used by the EDNAR Web Client, which includes configuration for things such as single sign-on (SSO). Configuration for setting up IIS hosting if not provided by Zepben.

Configuration​

The configuration for the EDNAR Web Client can be found in the /configs directory of both the main EDNAR Web App and the EDNAR Admin App files. In this folder, there will be a config.json file that must be used to populate the required config.

OptionDescriptionRequiredDefault
msalConfigObject holding MSAL config for SSO. More information at the MSAL Documentation.Only if SSO is enabled
msalLoginRequestObject holding MSAL Authorization URL Request config for SSO login. More information at the MSAL DocumentationOnly if SSO is enabled
msalSilentRequestObject holding MSAL Authorization URL Request config for SSO silent login. More information at the MSAL DocumentationOnly if SSO is enabled
msalLogoutRequestObject holding MSAL EndSessionPopupRequest config for SSO logout. More information at the MSAL DocumentationNo
useIdTokenFlag for whether the auth provider ID token should be used instead of the Access Token. If false, the access token will be used as user's auth token for EDNAR requests.Nofalse
useEmailAsLoginHintFlag for whether the user's email address should be used as their SSO "login hint". If false, the user's username will be used.Nofalse
displayDetailedAuthErrorsFlag for whether detailed authentication errors should display to users on the login page. Typically used for debugging, and disabled for production instances.Nofalse
adminBasePathBase path of the EDNAR Admin app. Used when the web client is served by the App Server.No
mapLayerColoursObject containing map layer colours that is used for styling the EDNAR network maps. Note that the full object must be included if this option is configured. Not including this option will fall back to the EDNAR defaults.No
