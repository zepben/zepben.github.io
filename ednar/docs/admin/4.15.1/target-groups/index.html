Target Groups

Notification target groups can be viewed using the admin app by navigating to the Configuration -> Notification Management -> Target Group page from the top menu. Each target group listed is displayed with their set target flags in the table as either a cross for not set or a tick for set.

A new target group can be created by using the Create Target Group button on the top right.

Target groups can include one or more specialised groups, usually related to the NAR that the notification is related to. These are listed down the page as individual toggles.

Target groups can also include specific participant types. The available participant types are listed on the left - these can be added by selecting one or more checkboxes, then pressing the right arrow in the center.

Once a target group is ready, hit the Save button in the top right corner. Existing groups can be edited by selecting Edit from the Target Groups list.
