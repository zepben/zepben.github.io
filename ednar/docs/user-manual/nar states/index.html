Nar States

STATESDESCRIPTION
UnsavedNAR ID has not been generated. NAR will not appear in the upcoming jobs page until saved. This status does not appear against the NAR.
DraftThis state is where information is gathered. The system is configured to require an applicant (requestor) to provide a minimum set of information before the NAR can be sent to the next state in the workflow. This ensures all required information is provided.
SubmittedThis state is where information is checked. While in this state, the information provided by the applicant is reviewed. An approver/reviewer of a NAR can send the NAR back to a draft state (reject the NAR) from the submitted state or can request more information from the applicant. This ensures all required information is accurate and that the work can go ahead
Pending SubmissionThis state is a special state that the NAR is sent to if an applicant picks a day that is within the NARs configured lead time. It is intended to support an additional level of approval before the NAR can be sent to the Submitted state.
AcceptedThis state is generally only used by the ‚ÄúQuick Access NAR Lifecycle‚Äù, however this state is also used in the DMS lifecycle if the integration with the DMS has failed. In both cases, this state indicates that the NAR has been approved and is being actioned.
DMS StatesThese states mirror the state of the NAR in the DMS, any of these states indicate the NAR has been approved and is being actioned in the DMS. These states are: DMS Created, DMS Submitted, DMS prechecked, DMS Approved, DMS Switching Commenced, DMS Switching Completed
CompletedThe NAR is sent to a completed state by a user if the work to access the network was completed successfully, and there is nothing further to do.
CancelledA NAR is cancelled either because it never made it out of a draft or submitted state, or because the network access did not go ahead as planned.
ReschedulingThis is the state the NAR is sent to if it is being re-scheduled. Only a limited number of fields in the NAR can be changed while in this state.
ResubmittedThis is the state the NAR is sent to after a NAR has been rescheduled. This is the state from which a rescheduled NAR is approved.
