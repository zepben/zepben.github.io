Planner Checklist

To add a Planner for the NAR, the planner panel must be in edit mode.

Adding a planner for the NAR can be done by entering a recognised name into the name field and clicking "Add". The planner chosen will be notified, any notes or messages for that planner can be sent with that notification by entering a message into the note field.

This can be done by clicking the view mode button. After this the planner's name can be entered and a note written for them to receive with the notification that they have been made planner.

These questions are configurable and will change depending on the processes of your company. This example contains three types of questions.

1. Clicking the Set button will set the answer to the question, with an information button that can appear when it is set to provide additional information.

2. Clicking "Yes" or "No" will answer the question, a "clear" button is also available if the question needs to be reset to an unanswered state.

3. Clicking the tick box will answer the question, the box can be ticked and unticked freely.

Any changes made to this checklist can be saved using the save button at the bottom of the checklist.
