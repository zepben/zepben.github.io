Quick Search

The Network Access Requests can be 'quick' searched by clicking on the quick search feature on the side bar. A field will then appear allowing for NARs to be searched by NAR ID or DMS ID that must be entered directly and exactly as the ID appears in the system.

To search for NARs using the Quick Search either the NAR ID, the DMS ID (J number) or the PoN ID (PoN project number) must be entered after the relevant tab has been selected.
