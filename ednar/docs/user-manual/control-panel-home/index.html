Control Panel Home

After the New Planned Request has been saved and submitted it must be accepted to take effect. To accept a NAR the control panel must be opened.

The control panel button will give access to the control panel which will allow for the review of the NAR.

The checklists on the left side of the control panel:

1. Clicking on the Planner Checklist opens the Planner Checklist panel.

2. Clicking on the OSSO Checklist opens the OSSO Checklist panel.

3. Clicking on the Controller Checklist opens the Controller Checklist panel.

4. Clicking on the Operator Checklist opens the Operator Checklist panel.

The Change State options on the right side of the panel:

5. After the checklists have been completed the submitted NAR can be accepted.

6. More Information Required returns the NAR state to draft with special conditions. That is the NAR can be resubmitted without changing the date and the NAR will go the accepted state immediately.

7. Clicking Recall is only available to the applicant who wrote the orginal NAR. Recall returns the NAR state to draft.

8. Clicking Decline returns the NAR to draft without any special conditions.

9. Clicking the Cancel button changes the state of the NAR to terminated which allows no more work to be completed on the NAR.
