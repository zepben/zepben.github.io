Introduction

EDNAR (Electrical Distribution Network Access Register) provides support for planned network access requests, managing the workflow, customer notification process, audit trail, attachments and internal notification processes.

This user manual provides detailed instructions on the use of the EDNAR web client for creating, tracking and managing Network Access Requests (NARs). An overview of EDNAR, its features and its functionality can be found in the EDNAR Feature Tour.

The EDNAR system administration guide can be found in the EDNAR Admin Documentation.

Contact

If you have any queries regarding this documentation, or EDNAR in general, please feel free to contact us.

Note

EDNAR is almost completely configurable - this manual covers the basic use and features of EDNAR, however other configurations may have slightly different workflows, UI text and functionality depending on how your organisation has configured it.
