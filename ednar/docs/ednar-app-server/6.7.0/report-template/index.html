NAR Summary PDF Report Template

The NAR PDF Report is produced by the EDNAR Reporting Service, and uses a HTML template file to determine the format, contents and styling of the produced PDF summary report. This template can be edited by administrators with a basic understanding of HTML and mustache template tags.

Editing the template​

To edit the template, the NarReportTemplate.html file located in the report-templates folder of the Reporting Service can be edited in any text editor. It is highly recommended to back this file up before making any changes.

The contents of the template can be edited using HTML and CSS to adjust the layout, contents and styling. To use fields from the NAR to populate data in the report, Mustache tags can be used to pull specific fields from the NAR into the template, that will then be populated from the NAR when the report is generated. Examples of the usage of these tags are included in the default NarReportTemplate.html file, but the basics are below.

Tag Usage​

There are 2 main ways to use the tags. The first and most convenient involves using opening and closing tags from the top-level field (e.g. nar), and then pulling the specific fields with individual tags within the open and closing tags:

{{#Nar}}
{{Id}}
{{/Nar}}

The second way involves specifying the full "path" of the specific field using ., and 3 sets of braces:

{{{Nar.DmsSwitchingDetails.DmsId}}}

For lists of values, using the opening/closing braces will populate each item in the list within the tags.

{{#Participants}}
{{Type.Name}}
{{User.First_Name}} {{User.Last_Name}}
{{/Participants}}
