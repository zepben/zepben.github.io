Criteria

Notification criterias can be viewed using the admin app by navigating to the Configuration -> Notification Management -> Criteria page from the top menu.

A new criteria can be created by using the Create Criterion button on the top right next to the search control.

A criteria can be composed by selecting whether All or Any conditions need to be met and subsequently adding conditions to the list. A condition can be the following :

  1. NAR question answer
  2. Built in question answer
  3. Existing criteria

Constructing complex criteria​

Example criteria​

(`REFCL Question 2` is answered as `Yes`) and (`Other Question 1 is answered as `Yes` or `REFCL Question 3` is answered as `Yes`)

To construct the criteria first answer All for the Should all or any of the conditions be met? question. Then add the following question answers using Add Condition modal :

  1. REFCL Question 2 with answer Yes

Following which add a group using the Add Group option. Answer Any against the Should all or any of the conditions be met? question and add the following question answer :

  1. Other Question 1 with answer Yes
  2. REFCL Quesiton 3 with answer Yes

After creating the criteria click on the Save button on the top right to save it.
