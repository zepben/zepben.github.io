Version: 1.1.0

Standalone Executor

This entry point maintains the behaviour of the previous opendss-jvm-executor container.

Config

Global config​

This entrypoint currently does not use any global configuration items.

KeyDescriptionRequired
ewbObject that holds ewb connection configNo
inputDbObject that holds inputDb connection configNo
loadDbObject that holds loadDb connection configNo
resultsDbObject that holds resultsDb connection configNo

JvmExecutionConfig​

KeyDescriptionRequired
rmqHostConfigObject that holds rmq connection configYes
modelQueueThe name of the queue to retrieve the openDSS model from on the configured rmq serverYes
resultsPrefixThe prefix of the stream to send the openDSS results on the configured rmq serverYes
dvmsConfigObject that holds DVMS configuration if DVMS is to be usedNo
workPackageIdThe workPackageId being executedNo
modelObject that holds information about the model being executedYes
storageBackendObject that holds StorageBackendConfiguration configuration for retrieving the model dataYes

JvmExecutionConfig​

KeyDescriptionRequired
feederThe feeder of the model being executedYes
loadTimeStartThe loadTimeStart of the model being executedYes
loadTimeEndThe loadTimeEnd of the model being executedNo
scenarioThe scenario of the model being executedYes
yearThe year of the model being executedYes

DVMS Config​

OptionDescriptionRequiredDefault
lowerLimitThe lower voltage limit in PU.Yes
upperLimitThe upper voltage limit in PU.Yes
lowerPercentileThe percentile value of customers to be cehcked against the PU limit.Yes
upperPercentileThe percentile value of customers to be cehcked against the PU limit.Yes
reportingLowerLimitThe lower voltage limit in PU used when calculating log output values. Defaults to lowerLimit.NoThe value of lowerLimit above.
reportingUpperLimitThe upper voltage limit in PU used when calculating log output values. Defaults to upperLimit.NoThe value of upperLimit above.
reportingLowerPercentileThe percentile value of customers to be cehcked against the PU limit used when calculating log output values. Defaults to lowerPercentile.NoThe value of lowerPercentile above.
reportingUpperPercentileThe percentile value of customers to be cehcked against the PU limit used when calculating log output values. Defaults to upperPercentile.NoThe value of upperPercentile above.
logVoltagesWhether to output the voltage information used by DVMS on each time step.Nofalse
maxDVMSIterationsMax number of tap changes to be made by DVMS per time step.No4
maxSolveRetriesThe number of times to retry solving a time step that openDSS reports didn't converge.No3
continueOnFailedToConvergeWhether to continue to the next time step after reaching maxSolveRetries with openDSS still reporting the model failed to converge.Nofalse
regulatorConfigObject holding configuration for the simulated regulator that maintains the average voltage outside of DVMS tap changes.No
requeueModelsWhether to requeue the model into the rmq queue once the model has completed running. Setting this to true leaves the model in rmq queue for reuse when debugging.Nofalse

Regulator Config​

OptionDescriptionRequiredDefault
puTargetTarget average voltage (of all LV loads) to maintain in PU.No1.0
puDeadbandPercentThe dead band to be used when maintaining average voltage, in percent. (puDeadbandPercent: 2.0, allows 1% either side of puTarget)No2.0
maxTapChangePerStepMax number of tap changes that change be performed but while trying to maintain puTarget per time step.No3
allowPushToLimitAllow the regulator to push customers/loads outside of lowerLimit/upperLimit while attempting to maintain puTarget. Up to lowerPercentile/upperPercentile customers total for there respective limitsNofalse

Example config:

{
  "global": {
    "ewb": null,
    "inputDb": null,
    "loadDb": null, 
    "resultsDb": null
  },
  "entrypoint": {
    "type": "standaloneExecutor",
    "config": {
      "rmqHostConfig" : {
        "host": "localhost",
        "port": 5672,
        "username": "guest",
        "password": "guest"
      },
      "workPackageId": "testWorkPackage",
      "resultsPrefix": "myTestResults",
      "model": {
        "feeder": "FDR01",
        "loadTimeStart": "2025-05-23T02:00:00Z",
        "loadTimeEnd": null,
        "scenario": "test-scenario",
        "year": 2025 
      },
      "dvmsConfig": {
        "lowerLimit": 0.94,
        "upperLimit": 1.1,
        "lowerPercentile": 1.0,
        "upperPercentile": 99.0,
        "logVoltages": true,
        "regulatorConfig": {
          "puTarget": 1.00,
          "puDeadbandPercent": 2.0,
          "allowPushToLimit": true
        }
      }
    }
  }
}
