Input Files

The input file name for the weather CSV readings should follow the following filename format:

WEATHERZONE_DATA_<YYYYMMDD>_<YYYYMMdd>

where <YYYY> <MM> <dd> are the year, month and date.

The following is a sample file name:

WEATHERZONE_DATA_20170831_20170904
