ContingencyEquipment

Class Description​

Equipment whose in service status is to change, such as a power transformer or AC line segment.

Attributes​

NameTypeDescription
contingentStatusContingencyEquipmentStatusKindThe status for the associated equipment when in the contingency state. This status is independent of the case to which the contingency is originally applied, but defines the equipment status when the contingency is applied.
