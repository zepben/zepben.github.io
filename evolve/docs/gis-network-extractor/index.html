Overview

The GIS Network Extractor builds an Energy Workbench network model from GIS json files in conjunction with other data sources. As EWB deals with LV data, it also uses an LV mapping file to map meters and supply points to transformers in the network model.

This tool will read the GIS JSON Network Data as well as multiple other sources of auxiliary information and converts it to an EWB Network.
