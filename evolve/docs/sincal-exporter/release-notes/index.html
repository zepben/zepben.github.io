v2.6.1#

Breaking Changes#
  • None.
Enhancements#
  • Made the rewiring of ISO transformers to look for direction not single/multi phasing.
  • Altering default text position to be offset from nodes; allows easier manipulation of network data for end users.
  • Now only adding broken bridges if they are closed.
  • Writing Flag_LineType for underground conductors.
  • Additional Sincal layer for 0 length conductors.
  • Style underground conductors in Sincal with broken lines.
  • Energy profile load flow.
  • Transformer name plate rating load flow.
  • Remove "cancelled by user" Prompt.
  • Prompt when export has finished.
  • Updated default sincal template with new colors for the layers.
  • Log date is now Australian date format.
  • Geographic location of nodes are written to make it easier for inserting background map.
  • Prepending of some nodes with FN to make it easier for fitting the diagram to the background map.
Fixes#
  • Corrected the handling of conductors with more than two terminals which resulted in excess buckle points being created.
  • Sincal exporter failing to deal properly when any feeder with a HV customer metering point and HV customer are connected in series.
  • Fixed an issue where the log is scrolling up and down with the export finished.
Notes#
  • None.

v2.6.0#

Breaking Changes#
  • None.
Enhancements#
  • Mapping data for zone infeeder attributes via CSV file.
  • Setting of zero seq. resistance and zero seq. reactance is now in default conductor options.
  • Support for single phase (2 core) exports to Sincal
  • Multi terminal conductor support.
  • You can now specify SINCAL database version to target.
  • You can now specify what voltage to override the network with.
Fixes#
  • None.
Notes#
  • None.

v2.5.1#

Breaking Changes#
  • None.
Enhancements#
  • None
Fixes#
  • Unfreezes the log when it is cleared.
  • Fixed log to not show anything when no filter is selected.
  • Fixed rendering issue on Windows 7 when filter button is toggled.
Notes#
  • None.

v2.5.0#

Breaking Changes#
  • None.
Enhancements#
  • Enhanced the messages when errors occur loading feeders.
  • Added object counts to database creation log entries.
  • Added layers and styling to highlight conductors with missing length and missing or suspect types.
  • You can now specify the default values to use for conductors with missing types.
  • The log has been enhanced to add colour and filtering. The log will be filtered by default to only show errors and warnings.
  • You can now chose your own sin file to use as the template.
Fixes#
  • Corrected transformer name formatting.
  • Fixed the full debug log message describing which breakers have been excluded.
  • Added support for GIS based load ids.
  • No longer removing the entire feeder when the infeed is the feeder CB.
Notes#
  • None.

v2.4.0#

Breaking Changes#
  • None.
Enhancements#
  • You can now export a geographic view of the model.
  • All selected feeders will be shown regardless of the filter.
  • A node is now placed for every breaker making them easier to search for.
  • Updated the names of loads and ISO transformers to allow them to be added to the smart load overrides.
Fixes#
  • Nodes are now being exported with the correct symbol type.
  • No longer fails to export when all selected items have no graphics.
  • Improved some of the process log messages.
  • Fixed spelling mistakes in some log message.
Notes#
  • None.

v2.3.0#

Breaking Changes#
  • None.
Enhancements#
  • Breakers with incorrect wiring are now processed rather than failing the export. The wiring may be incorrect and should be checked.
  • Regulators are now collapsed when a fault indicator is attached to one of the cables running into the regulator.
  • Removed warnings about bends in conductors.
  • Changed the names of nodes to be more user friendly.
  • Feeder CB nodes now have a short name which can be displayed to label each feeder.
  • Zone substations now have a substation node with a short name which can be displayed.
  • Zone substations components between FCB's and FSP's are now removed.
  • Missing graphics can now be interpolated.
Fixes#
  • None.
Notes#
  • None.
