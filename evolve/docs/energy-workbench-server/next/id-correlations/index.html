ID Correlations

The id correlations is specified in a JSON file with filename format : <YYYY-MM-DD-id-correlations.json> under a single date subdirectory.

KeyDescription
systems
cores
cores#
OptionDescription
id
<system_id_1>
<system_id_2>

Sample Id Correlations#

{
"systems": ["GIS", "DMS"],
"cores": [{
"id": "ID_1",
"GIS": ["GIS_ID_1"],
"DMS": ["DMS_ID_1"]
}, {
"id": "ID_2",
"GIS": ["GIS_ID_2"],
"DMS": ["DMS_ID_2"]
}]
}
