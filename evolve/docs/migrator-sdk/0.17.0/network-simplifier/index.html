Network Simplifier

The NetworkSimplifier simplifies a CIM network model to prepare it for direct translation to a bus-branch model (e.g. OpenDSS).

  • Only certain types of equipment are kept: lines, transformers, energy sources, energy consumers, and power electronics connections.
  • Switches may be included in the simplified network by setting the keepSwitches parameter to true. Otherwise, closed switches are collapsed and open switches are removed.
  • All other equipment, such as junctions, are treated as having zero impedance and are collapsed.
  • Strings of connected lines sharing a common impedance, phasing, and wire info are collapsed into a longer line. The length of this line is equal to the sum of the lengths of the original lines.
  • Networks below SWER isolation transformers and distribution transformers may also be optionally collapsed.
  • Feeder head terminals are relocated to an appropriate nearby terminal if the original terminal is collapsed as part of a group of excluded equipment.
  • To make translation to certain models easier (e.g. OpenDSS), each terminal is guaranteed to be connected to a connectivity node.

Usage​

Create a NetworkSimplifier and process a NetworkService.

  • Kotlin
val networkSimplifier = NetworkSimplifier()

networkSimplifier.process(networkService)

Create a NetworkSimplifier that keeps switches in the simplified model.

  • Kotlin
val networkSimplifier = NetworkSimplifier(keepSwitches = true)

networkSimplifier.process(networkService)

Create a NetworkSimplifier that uses the current state of the switches, rather than their normal state.

  • Kotlin
val networkSimplifier = NetworkSimplifier(networkState = NetworkState.Current)

networkSimplifier.process(networkService)

Create a NetworkSimplifier with a different threshold for line resistance and reactance. Lines that fail to exceed either threshold are treated as if they had zero impedance, and are collapsed along with adjacent zero-impedance equipment.

  • Kotlin
val networkSimplifier = NetworkSimplifier(minLineResistanceOhms = 0.002, minLineReactanceOhms = 0.003)

networkSimplifier.process(networkService)

Create a NetworkSimplifier that collapses both SWER and distribution networks.

  • Kotlin
val networkSimplifier = NetworkSimplifier(collapsedSwerVoltage = 250, collapseLvNetwork = true)

networkSimplifier.process(networkService)
