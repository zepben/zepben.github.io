v1.1.0

Breaking Changes
  • None.
New Features
  • Added export to CSV/JSON functionality for data displayed on a combined drill down graph.
  • Added tabular view for a combined drill down graph which lists supply points that compose the profile. You can select/deselect supply points to recalculate the profile.
  • Added export to CSV/JSON functionality for recalculated combined drill down graph.
Enhancements
  • The combine page now supports setting graph options (showKva, showAmps, showDuration, showY0).
Fixes
  • Fixed the rendering of rating series in the weather, max demand, min demand and date profile pages when adding additional series to the graph.
  • Fixed a bug where the URL parameter "systemTag" was not being used correctly in the weather, max demand, min demand, date and 3D profile pages when changing tabs.
  • Fixed a bug where the rating series would disappear when selecting the option to display load in Amps.

  • None.
