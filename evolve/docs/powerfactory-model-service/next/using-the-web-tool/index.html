Using the Web UI

This guide walks you through generating PowerFactory models from the Network Explorer web interface.

Accessing the Model Builder​

  1. Navigate to the Power Factory tab in the top navigation bar.

  2. Click the Power Factory dropdown to see your options:

    • List Models: View previously generated models
    • Export new Model: Create a new PowerFactory model

  1. Select Export new Model to get started.

Selecting Your Network​

The Model Builder opens with a hierarchical view of your network.

Network Selection Options​

Show Low voltage: Toggle whether to include LV networks in your model. Selecting 'No' will only show HV and MV elements.

Search: Quickly find specific assets or locations in your network.

Selecting Network Elements​

Choose what to include from the network hierarchy:

  1. Network-wide: Check the top-level "Network" box to export everything
  2. Region/Metro: Expand and select specific areas
  3. Zone Substation: Select individual substations (like "KENTLYN" in the example)
  4. Feeder: Pick specific feeders within a substation

Navigation Tools:

  • Reset: Clear your selections and start over
  • Pan To Selection: Center the map on your selected elements

Configuring Export Options​

Before exporting, you'll need to configure how your model gets built.

Click the Options button. If you haven't created any export profiles yet, you'll see a prompt to add one.

Creating an Export Profile​

Export profiles save your configuration settings so you can reuse them for different models.

  1. Click Add Option
  2. Give your profile a name
  3. Click Add

Configuring Export Settings​

Your profile has three tabs of settings:

Transformers Tab​

Distribution Transformer Internal Grounding Impedance, LV Side

Set the resistance and reactance values (both in Ohms) for distribution transformer grounding. These values apply to all distribution transformers in your model and are important for accurate fault analysis and load flow studies.

Loads Tab​

Load Configuration

Choose how loads appear in your model:

  • Spread max demand: Uses maximum demand readings from smart meters as static loads throughout the network. This is the standard choice for steady-state load flow studies.

  • Timeseries connection: Links smart meter time-series profiles directly to the model, enabling dynamic load flow studies across different time periods. This option requires your Energy Workbench instance to have timeseries data configured. Note: This option is currently in Beta and may require additional setup. Contact Zepben support for assistance.

Forecast Tab​

Availability

This option isn't available for all users. Contact Zepben support to enable forecast exports based on Hosting Capacity Module (HCM) scenarios.

Check Enable Forecast to include hosting capacity forecast scenarios in your export.

Configure these parameters:

  • Scenario ID: Select the forecast scenario from your Hosting Capacity Module results
  • Years: Comma-separated forecast years
  • PV Upgrade Threshold (W): Minimum PV system size that triggers an upgrade
  • BESS Upgrade Threshold (W): Minimum battery storage size that triggers an upgrade

This lets you export multiple PowerFactory model variants representing different years and DER penetration scenarios from your hosting capacity analysis.

Saving Your Profile​

After configuring your settings, click Save. You can create multiple profiles for different use cases—like one for HV-only models, another with LV networks, or one for forecast scenarios.

Generating the Model​

With your network selected and options configured, you're ready to generate the model.

Final Settings​

Is Public: Choose whether other users in your organization can view and download this model.

File Name: Give your model a descriptive name.

Export Option: Select the profile you just configured. You can also create new profiles here using the Add button.

Open Model list after hitting submit button?: Choose Yes to jump to the model list after export starts, or No to stay on the current screen.

Starting the Export​

Click Submit to start generating your model.

The export runs in the background and may take several minutes depending on your network size. Feel free to navigate away while it's running.

Managing Your Models​

Head to Power Factory > List Models to see all your generated models.

Model List Overview​

The list shows:

  • Id: Unique identifier for each model
  • Name: The filename you specified
  • Created At: When the model was generated (click to sort)
  • Actions: What you can do with each model

Model List Type Filter: Filter by visibility:

  • Both: All models (public and private)
  • Public: Models shared with your organization
  • Private: Just your models

Sorting: Click "Name" or "Created At" headers to sort ascending or descending.

Model Actions​

Each model has four actions:

Download: Download the completed DGS file.

If export failed, you'll see a red X instead. Hover over it to see what went wrong.

Common errors include connection timeouts (usually fixable by regenerating), configuration issues, or data availability problems.

Delete: Permanently remove the model.

Request Information: View the exact configuration used to generate this model, including which network elements were selected and what export options were applied. Useful for reproducing models or troubleshooting differences between exports.

Regenerate: Recreate the model with the same configuration. Handy when an export failed due to a temporary error, or when you want to refresh a model with updated network data.

Opening Your Model​

Once downloaded, open the DGS file in DIgSILENT PowerFactory to start your analysis. The model includes all selected network elements with your configured settings, ready for load flow studies, protection coordination, or other power system analyses.

For further assistance, please contact Zepben support.
