Output Tables

Network Performance Metrics Enhanced​

This table provides the core results of a work package, summarised by time of day and season. It includes key identifiers for the conducting equipment, detailed voltage metrics, load and consumption figures, generation and curtailment data, network constraint indicators, unserved energy calculations, and customer reliability values. This comprehensive dataset enables in-depth analysis of network performance and constraint impacts across different operating conditions.

Full Table: network_performance_metrics_enhanced ▼

Weekly Reports​

This table provides weekly voltage compliance reports, following the ESC (VIC) methodology. It enables forecasting and assessment of network voltage compliance, using Saturday-to-Saturday periods for V1 and V99 percentile calculations.

Full Table: weekly_reports ▼

Constraints​

This table provides a comprehensive list of all network constraints detected during the simulation, with each row representing a distinct voltage or thermal exceedance event. It enables users to identify when and where network limits were breached, the type of constraint (e.g., voltage high/low, thermal overload), and key metrics such as the duration, magnitude, and associated energy or value at risk. This information supports detailed analysis of constraint frequency, severity, and potential mitigation strategies.

Full Table: constraints ▼

Duration Curves​

This table presents duration curves for each measurement zone, displaying power (kW) across the full modelling period, sorted in 1% increments. Duration curves provide a visual summary of network utilisation patterns, highlighting periods of peak and low usage. They are useful for identifying spare capacity, assessing the frequency and magnitude of constraints, and evaluating the potential benefits of interventions such as battery energy storage systems (BESS) or network upgrades. For constrained measurement zones, the shape of the curve can help determine whether solutions like BESS (to address short-duration peaks) or traditional network augmentation (for sustained overloads) are likely to be most effective.

Full Table: duration_curves ▼

Overloads Raw​

This table contains raw asset-level overload data, providing detailed information for each asset, phase, and timestamp. It is useful for identifying and reporting thermal constraints at the asset level, supporting in-depth analysis of overload events and their causes.

Full Table: overloads_raw ▼

Voltage Exceptions Raw​

This table provides detailed raw voltage exception data at the node level for each modelled timestep. It lists all nodes within a feeder that experience voltage extremes, including the magnitude and location of minimum and maximum voltages, as well as counts of overvoltage and undervoltage events for both HV and LV networks. This enables granular analysis of voltage compliance issues and identification of problematic locations within the network.

Full Table: voltage_exceptions_raw ▼

Metrics Enhanced Profile Raw​

This table contains detailed raw data for the enhanced metrics, providing a time series view of network performance at each interval. It enables in-depth analysis of constraint events, load and generation patterns, and supports assessment of BESS dispatch or other interventions required to address network issues.

Full Table: metrics_enhanced_profile_raw ▼

Link to Input​

This table links each output record to its corresponding input configuration, enabling users to trace which input profiles, DER allocations, and scenarios were used for each work package result. This supports reproducibility and transparency in analysis.

Full Table: link_to_input ▼

Logs​

This table contains a log of all errors, warnings, and informational messages generated during the execution of a work package. It enables users to trace issues, monitor process steps, and review system events for troubleshooting and auditing purposes.

Full Table: logs ▼

Network Performance Metrics​

This table contains the basic (non-enhanced) network performance metrics. It is retained for backward compatibility but has been superseded by the network_performance_metrics_enhanced table, which provides a more comprehensive set of results and metrics. Users are encouraged to transition to the enhanced table, as this table may be removed in a future version.

Full Table: network_performance_metrics ▼

Energy Meters Raw​

This table contains detailed raw data from energy meters, providing interval-level measurements of real and reactive power, losses, and delivered energy for each measurement zone. It supports in-depth analysis of feeder performance, loss allocation, and validation of voltage regulation or network interventions. Use this table for advanced diagnostics and to trace the flow of energy and losses across different voltage classes within the network.

Full Table: energy_meters_raw ▼

Energy Meter Voltages Raw​

This table provides detailed raw voltage data from energy meters at each modelled timestep and measurement zone. It enables in-depth analysis of voltage regulation, phase imbalances, and the impact of network interventions. Use this table for advanced diagnostics, validation of modelled voltages against field data, and to investigate voltage performance across all phases and voltage groups within a feeder.

Full Table: energy_meter_voltages_raw ▼

Calibration Meter Results​

This table provides the detailed results from the calibration process, including real and simulated voltage measurements, differences, and validity flags. It is an output of the Model Calibration process.

Full Table: calibration_meter_results ▼
