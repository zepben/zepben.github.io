How to make a new page​

If you want to add a new page, ie a new how to-guide or a new tutorial:

  • create a new markdown file
  • add the block as above, giving the page a unique id. The file name doesn't have to match, but it helps to keep everything easy to understand the 'slug' is the url it will have. unless otherwise desired, use the id.
  • add that id, including its folder structure, to the sidebar.js

For more info, see here: https://docusaurus.io/docs/docs-introduction Note that as of 13/5/25, Zepben is using Docusaurus 2.3.1, which is well behind the current version. Care should be taken when updating this guide to use things that work in v2.3 of Docusaurus.
