Version: 0.8.0

Overview

CLI Debug Configuration​

The following options are passed via CLI arguments.

OptionDescriptionTypeRequired
--debug-save-pathIf specified, results for the fetched SYF will be collected into this file for later debugging rather than processed. Incompatible with --debug-host-path.File pathNO
--debug-host-pathIf specified, results for the fetched SYF will be fetched from this file rather than RabbitMQ. Incompatible with --debug-host-path.File path (must exist)NO
--result-processor-configIf specified, the resultProcessorConfig will be sourced from this JSON file rather than the SYF from RabbitMQ.JSON File path (must exist)NO

Configuration From SYF​

The following options can be passed via the SYF message in the resultsProcessorConfig parameter (encoded JSON object). These are the same parameters that can be specified via the hosting capacity service configuration file.

OptionDescriptionTypeRequired
enhancedMetricsConfigCollection of flags controlling the enhanced metrics outputJSON ObjectNO
Enhanced Metrics Config​

These flags control which of the enhanced metrics tables will be populated in the results database.

OptionDescriptionTypeRequiredDefault
populateEnhancedMetricsIndicates the enhanced_metrics table should be populatedbooleanNOfalse
populateEnhancedMetricsProfileIndicates the enhanced_metrics_profile table should be populatedbooleanNOfalse
populateDurationCurvesIndicates the duration_curves table should be populatedbooleanNOfalse
populateConstraintsIndicates the constraints table should be populatedbooleanNOfalse
populateWeeklyReportsIndicates the weekly_reports table should be populatedbooleanNOfalse
calculateNormalForLoadThermalIndicates thermal exceedances (load) should be detected using the normal ratingbooleanNOfalse
calculateEmergForLoadThermalIndicates thermal exceedances (load) should be detected using the emergency ratingbooleanNOfalse
calculateNormalForGenThermalIndicates thermal exceedances (generation) should be detected using the normal ratingbooleanNOfalse
calculateEmergForGenThermalIndicates thermal exceedances (generation) should be detected using the emergency ratingbooleanNOfalse
calculateCO2Indicates the value of avoided CO2 emissions should be included in $ per kWh calculationsbooleanNOfalse

Example​

{
  "enhancedMetricsConfig": {
    "populateEnhancedMetrics": true,
    "populateEnhancedMetricsProfile": true,
    "populateDurationCurves": true,
    "populateConstraints": true,
    "populateWeeklyReports": true,
    "calculateNormalForLoadThermal": true,
    "calculateEmergForLoadThermal": true,
    "calculateNormalForGenThermal": true,
    "calculateEmergForGenThermal": true,
    "calculateCO2": true
  }
}
